# Faircorp Vue Application

## Setup
1. npm install -> to install the node modules
2. npm run serve -> to run the application.

## Components

1. Building
2. Developer
3. Heater
4. Hello
5. Home
6. Location Map
7. Room
8. Weather (the API is very unstable, please read the code)
9. Window

## Dependencies 
1. Axios
2. Bulma
3. core-js
4. vue-browser-geolocation
5. vue-googlemaps
6. vue-router
